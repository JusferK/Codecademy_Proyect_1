    Basic structure

But if I can explaing that using my words, is basically the skeleton for a web page, like literally a skeleton. In order for us to create a web document... because that what it actually is, you need to use the following tags: <html>, <head>, <body>. Those are the main tags and without them, the html structure would actually have some issues trying to build the page. Every tags start with the conditional mark less than < and closes with the conditional greater than > so in order to start a tag it would be something like <tag> and then close the tag just like the opening pattern but using a division mark withing the conditional marks... just like this: </> so the full tag would look like this: <tag></tag>.     Media queries

    What does it means media queries? what is that? how do eat that? Media queries are a fundamental component of responsive web design in CSS. They allow you to apply different styles to your web page based on various characteristics of the user's device, such as screen size, resolution, orientation, and more. Media queries enable you to create layouts and designs that adapt and look good on a wide range of devices, from desktop computers to smartphones and tablets.

    Syntax

    Media queries are typically written using the @media rule followed by one or more conditions enclosed in parentheses. The conditions specify when the styles within the media query should be applied.

    Conditions

    Media queries are based on various conditions. The most common condition is the screen width, but you can also use conditions for things like screen height, aspect ratio, orientation, and more.

    Using media queries

    You can place media queries in your CSS code to target specific screen sizes or other conditions. When a condition is met, the styles inside the media query block will be applied, and when the condition is not met, those styles won't affect the rendering of the page.

    Combining Conditions

    You can combine multiple conditions in a media query using logical operators (and, not, only, etc.) to create complex rules.

    Viewport Units

    Media queries are often used in conjunction with viewport-relative units like vw (viewport width) and vh (viewport height) to create flexible and responsive designs. Let me show how it would look like in code:

    .nav {
     background-color: #333;
     color: #fff;
    }

    @media (max-width: 768px){
     .nav {
      background-color:#777;
     }
    }

    @media (max-width: 480px){
     .nav {
      background-color:#777;
     }
    }
